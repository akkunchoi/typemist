Typemist: augments minimist with TypeScript and decorators.

## Usage

See src/sample.ts

## Supported API

- `@Paramter(number)`
- `@Option`
- `@Description(string)`

## Requirement

```
{
    "compilerOptions": {
        "experimentalDecorators": true,
        "emitDecoratorMetadata": true
    }
}
```
